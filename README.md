# 🎬 AI Movie Trailer Generator

## Overview

The AI Movie Trailer Generator is an AI-powered application that creates cinematic movie trailers from a simple text prompt. The project combines multiple AI technologies to generate a trailer script, narration, visuals, and a final edited trailer video.

This project was developed using Python in Google Colab as part of an AI assignment.

---

## Features

* Generate movie trailers from text prompts
* AI-generated trailer narration
* AI-generated visuals
* Automatic video creation
* Background music integration
* Export trailer as MP4

---

## Technologies Used

* Python
* Google Colab
* Google Gemini API
* MoviePy
* FFmpeg
* PIL (Python Imaging Library)

---

## Project Workflow

1. User enters a movie idea.
2. Gemini generates a cinematic trailer script.
3. AI generates visuals for each scene.
4. Narration is created.
5. Visuals, narration, and music are combined.
6. Final trailer is exported as an MP4 video.

---

## How to Run

1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Replace `"YOUR_API_KEY"` with your own Gemini API key.
4. Run all notebook cells.
5. Enter a movie prompt.
6. Download the generated trailer.
